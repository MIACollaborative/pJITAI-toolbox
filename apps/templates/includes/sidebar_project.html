Why use the pJITAI toolbox?

This toolbox helps you design a personalizing Just-in-Time Adaptive Intervention (pJITAI) component, i.e., the toolbox helps you configure an algorithm that personalizes the intervention delivery of this JITAI component.

What is Personalization?

Data accruing from an individual may indicate that intervention option A is more effective than intervention option B when the individual is in certain states. A personalizing JITAI will increase the probability of sending intervention option A in these states.

For example, if an individual is more responsive to intervention option A during weekends, the JITAI will send intervention option A more frequently than intervention option B during weekends.

When can you use the pJITAI toolbox?

  • Scenario 1: When you have access to published results from a micro-randomized trial (MRT), the toolbox helps you find the information needed to configure a personalizing JITAI.
  • Scenario 2: When you have access to the data from an MRT of an intervention component you want to personalize, the toolbox guides you to run analyses to configure a personalizing JITAI.

Interested in practicing?

You can practice using this tool with HeartSteps, an mHealth intervention for physical activity. You can find a demo pJITAI (designed for HeartSteps) in "Add New Project" in the left-hand menu.

Some details about HeartSteps:

  • Goal: Provide an actionable message to help a person take a short walk in the near future.
  • Intervention options: (A) Send an activity suggestion; (B) Do not send an activity suggestion
  • Proximal outcome: The 30-minute step count after each decision point

How to use the pJITAI toolbox?

We suggest that if you are not familiar with our toolbox, you start by going through our HeartSteps demo pJITAI in "Add New Project" in the left-hand menu. (If you don't see a menu on the left, you may see a menu-like icon in the top right corner of your screen. Click it to open the menu.)

When you’re ready to use this tool to develop personalizing JITAIs for your study, you’ll configure a personalization algorithm for each intervention component separately.

For example, in the HeartSteps study, researchers were interested in personalizing a number of intervention components including “activity suggestion” and “planning.” In the demo, we show you how to design a personalization algorithm for the activity suggestion.

Once you finish designing your personalizing JITAI...

The toolbox will generate a URL for accessing the algorithm you just configured via an API so your app developers (e.g., Care Evolution and Fitabase) can integrate the algorithm into the code for your pJITAI. The algorithm can then begin to personalize sending of interventions. (More information can be found in FAQs.)
