Intervention component name

Provide the name of the intervention component that will be personalized by the RL algorithm. An intervention component is any part of a JITAI that has its own decision rule, for example the component “Morning Motivational Message” might be the name of a component designed to send participants a motivational message at 9am.

HeartSteps Example

For the running example, we will be using the “Activity Suggestions” intervention component from HeartSteps. This is a component that may send an activity suggestion via push message to each participant at any of 5 times per day (morning, midday, afternoon, evening). To follow the example, you can enter “Activity Suggestions RL” into the project name and “Activity Suggestions” into the “Intervention component name” field.
