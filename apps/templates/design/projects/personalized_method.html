General Settings / Personalization Method

This version of Thompson Sampling uses a Bayesian linear regression to model the relationship between the proximal outcome, states, and intervention options. This method determines which intervention option to deliver based on the posterior probability that the intervention option is most effective in the current state. To learn more, please check here [link to the tutorial].

Key assumptions:

  • There are two intervention options.
  • The proximal outcome is continuous.

In addition, in this version, the algorithm uses only data from each individual to learn and update that individual’s coefficients. (In other words, in this version, to personalize the decision rule for a participant, the algorithm does NOT use other participants' data.)

Coming soon: a version of Thompson Sampling for categorical proximal outcomes.

  • Explanation

Personalization method refers to the statistical/computational method (i.e., algorithm) that enables the personalization in a ”Personalizing JITAI.” This version of pJITAI implements a specific version of the Thompson Sampling algorithm. Please read the blue box for more information. Other methods (e.g., Thompson Sampling for categorical proximal outcomes) will be incorporated soon.
