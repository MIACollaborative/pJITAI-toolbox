What is the pJITAI Toolbox?

This toolbox is a tool to help you design a personalizing Just-in-Time Adaptive Intervention (pJITAI) for your study. It helps you design a reinforcement learning algorithm for a pJITAI component.


Where should I begin with?

Click on the "Getting Started" tab in the left-hand menu to find a step-by-step guide on how to use this toolbox to design a personalizing JITAI for your study.


Prerequisite: Bring MRT study data

For this version of pJITAI Toolbox, we assume you have a previously run MRT data. You will need this data to design a pJITAI. Please make sure your MRT data is ready before you start using the toolbox.
