{% set block1="Y" %} {% set block1_title=["Prior for the variance of the error term, ε"] %} {% set block1_text=["The prior distribution of the error variance in the Bayesian Linear regression is a “scaled inverse chi-square distribution.” The “scale” is denoted by σ02 and the “degrees of freedom” are denoted by ν in the above formula."] %} {% set block2="Y" %} {% set block2_title="How to set prior values?" %} {% set block2_text %} This discussion assumes that the proximal outcome is approximately standardized. Values for the scale and the degrees of freedom define your "prior belief" about the variance, σ2 of the error term (ε).
  • Degree of Freedom, ν, (Recommended: 5)
    This setting controls the model’s sensitivity to outliers (unusual proximal outcomes).
    • A high value for the degrees of freedom implies that the error term in the linear regression has close to a Normal distribution with σ2 approximately equal to σ02. However, if there are likely to be unusual error terms (i.e. outliers resulting from unusual values for the proximal outcome such as some unusually high 30 minute step counts in the case of HeartSteps) then it is better to choose a lower value of ν.
    • Setting the degrees of freedom to a lower value like 5 means that the error term in the linear regression has a distribution closer to a Student's t-distribution. This distribution has "heavy tails", meaning it expects some outliers and automatically down-weights their influence, leading to a more robust statistical analysis if there are outliers.
  • Scale, σ02
    For the scale, you can either use the average of the person-specific error variances from your MRT analyses or since the proximal outcome is approximately standardized you can use σ02=1.
