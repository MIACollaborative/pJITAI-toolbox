That is the terminology used throughout the pJITAI toolbox."] %} {% set block2="Y" %} {% set block2_title="Transforming the proximal outcome in HeartSteps" %} {% set block2_text %} Recall that in HeartSteps, the step count in a 30-minute window following each decision point is the proximal outcome. This 30-minute step count takes on integer values (0, 1, 2,...) and is heavily right-skewed with many values equal to zero and a long tail of higher step counts stretching towards the maximum expected value. Thus, to transform this proximal outcome so that its distribution is more symmetric, the raw step counts are log-transformed, like so: transformed_prox_outcome = log(raw_steps + 0.5) The value of 0.5 is added to the raw_steps value because it avoids taking the log of 0, which is undefined. If we assume 0 to be the minimum value for the raw 30-minute step count and 3000 to be the maximum value (no Usain Bolts were observed in HeartSteps V1 (link) data), we end up with: min_prox_outcome = log(0 + 0.5) = -0.7 max_prox_outcome = log(3000 + 0.5) = 8.0 Following this log transformation, the proximal outcome needs to be approximately standardized for the algorithm. Standardization transforms the (transformed) proximal outcome so that it has an approximate mean of 0 and an approximate standard deviation of 1. The formula for standardization is: Ystandardized = (Ytransformed - μY) / σY where:
  • Ytransformed is the log-transformed proximal outcome.
  • μY is the sample mean of the transformed proximal outcome.
  • σY is the sample standard deviation of the transformed proximal outcome.
The original HeartSteps implementation did not standardize the proximal outcome. However, in order to standardize the proximal outcome for HeartSteps for this illustration, we can use the rough rule of thumb: the average value of the (transformed) proximal outcome is roughly the center of this range (use (8.0-(-0.7))/2=4.4) and the standard deviation is roughly the length of the range divided by 4 (use (8.0-(-0.7))/4=2.2). Using those values, the minimum and maximum value for the standardized proximal outcome for HeartSteps will be -2.3 (use (-0.7 - 4.4)/2.2) and 1.7 (use (8.0 - 4.4)/2.2) respectively. {% endset %} {% include 'includes/resources.html' %}
